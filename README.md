🌐 Sitio Web Básico (HTML, CSS y JavaScript)
📌 Descripción

Este repositorio contiene un sitio web básico desarrollado con HTML, CSS y JavaScript, cuyo propósito es servir como guía práctica para estudiantes en el aprendizaje de la estructura y construcción de páginas web.

Incluye múltiples páginas, recursos multimedia y una organización de carpetas adecuada para proyectos web iniciales.

📦 sitio-web
├── 📁 .vscode
├── 📁 css
│   └── estilos.css
├── 📁 img
│   └── (imágenes del sitio)
├── 📁 video
│   └── (videos del sitio)
├── ayuda.html
├── contacto.html
├── index.html
└── README.md

🧩 Páginas del sitio
index.html → Página principal
ayuda.html → Página de ayuda para el usuario
contacto.html → Página con formulario de contacto
🎨 Tecnologías utilizadas
HTML5 → estructura del sitio
CSS3 → estilos y diseño visual
JavaScript → interacción (formularios y eventos básicos)
🚀 Cómo usar este proyecto
1. Clonar el repositorio
git clone https://github.com/USUARIO/NOMBRE-REPOSITORIO.git
2. Abrir en Visual Studio Code
Abrir la carpeta del proyecto
Ubicar el archivo index.html
3. Ejecutar el sitio
Clic derecho sobre index.html
Seleccionar "Open with Live Server" (si está instalado)
📚 Objetivo pedagógico

Este proyecto permite a los estudiantes:

Comprender la estructura de un sitio web
Organizar archivos y carpetas correctamente
Integrar HTML, CSS y JavaScript
Implementar navegación entre páginas
Insertar imágenes y videos
📝 Recomendaciones
Mantener nombres de archivos claros y sin espacios
Organizar correctamente las rutas (css, img, video)
Validar el código HTML y CSS
Usar comentarios para documentar el código
